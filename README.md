# ESP8266 Webserver project

In this project, we using:

- WebServer
- WebSockets
- mDNS
- DHT11
- Timer

# Features

- Control builtin LED from web page
- Sync LED state crossing all devices
- Showing temperature and humidity from DHT11 to web page
